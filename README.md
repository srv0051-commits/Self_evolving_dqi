# Self-Evolving Data Quality Intelligence System

A Python-based tool that automatically validates and monitors data quality 
in machine learning pipelines before model training begins.

## What It Does
- Detects missing values, schema mismatches, and outliers in datasets
- Generates structured reports flagging data quality issues
- Reduces manual data inspection by automating pre-processing validation

## Tech Used
- Python
- Pandas
- SQL

## Why I Built This
Poor data quality is one of the biggest causes of ML model failure. 
This tool acts as a quality gate before any dataset enters a training pipeline.

## How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python main.py`
